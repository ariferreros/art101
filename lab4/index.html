Task 1: Describing a Simple Task

  1. Set up a drawing utensil and something to draw on.
  2. In the center of your canvas draw a circle.
  3. Then at the top right and top left of your circle draw two upside down ‘V’s.
  4. Then return to the center of the circle and draw a ‘W’.
  5. Place a dot at the peak of the ‘W’.
  6. Place two dots to each side and slightly above the ‘W’.
  7. Finish by adding three lines directly next to the ‘W’ on each side. Like this: ⚞⚟

Task 2: Planning out a Game

  1. Print the background
  2. Randomly print the items for the character to pick up
  3. Print the character
  4. Wait for user input
  5. Handle movement:
    • W → move up
    • A → move left
    • S → move down
    • D → move right
  6. If user and item touch, remove item
  7. If no more items, display win message
